### cpms-java项目的前端用户角色权限部分

后端代码地址：https://github.com/gulang12/cpms-java

```bash
# 克隆项目

# 进入项目目录
cd cpms-java-vue

# 安装依赖
npm install

# 建议不要直接使用 cnpm 安装以来，会有各种诡异的 bug。可以通过如下操作解决 npm 下载速度慢的问题
npm install --registry=https://registry.npm.taobao.org

# 启动服务
npm run serve
```

# 构建生产环境打包文件
npm run build:prod
```
